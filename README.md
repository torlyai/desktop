## Auto-Updates

TorlyAI Desktop includes automatic updates. When a new version is available, you'll see an update notification in the app. You can also check manually via **Settings > About > Check for updates**.

## Features

- **Setup Wizard** — 8-step guided onboarding with AI team introduction
- **Autopilot** — Autonomous multi-agent workflow orchestration
- **Project Workspace** — Kanban board with AI-native task management
- **Business Plan Generation** — 7 comprehensive sections using specialized AI agents
- **Innovatorly Matrix** — 4F Formula evaluation for visa endorsement readiness
- **Financial Modeling** — 3-5 year projections with sensitivity analysis
- **Content Pipeline** — Living business plan assembly from agent outputs
- **Mermaid Diagrams** — Flowcharts, sequence, ER, and class diagrams in chat
- **File Previews** — PDF, images, JSON, HTML, and data tables
- **Multi-LLM Support** — Anthropic Claude, OpenAI, GitHub Copilot, OpenRouter, Ollama
- **Chinese Language** — Simplified Chinese full UI localisation

## System Requirements

### macOS
- **Minimum:** macOS 12.0 (Monterey), 4 GB RAM, 500 MB disk
- **Recommended:** macOS 13.0 (Ventura) or later, 8 GB RAM, 1 GB disk

### Windows
- **Minimum:** Windows 10 (64-bit), 4 GB RAM, 500 MB disk
- **Recommended:** Windows 11, 8 GB RAM, 1 GB disk
